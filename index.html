Litecore

A robust and powerful custom Litecoin Node for building and developing awesome Blockchain applications.

npm install -g litecore

Built on Litecoin

Litecore is a custom full litecoin node which means your apps run directly on the p2p network, providing a fast and responsive performance and platform for all the work you do.

Litecore is fully open source.
